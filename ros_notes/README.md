# ROS 2
versjon foxy.


## drift av ROS 2
start med å kjøre:

```Bash
source /opt/ros/foxy/setup.bash
```

Ellers vil ikke kommandoen `ros2` fungere.  
Den kan selvfølgelig appendes til `~\.bashrc` for å være kjørt hver gang, men der god trening å være bevist på dette fra starten av.
